# cli-pt

Chromium cli dictionaries opener for searching portuguese words

### Note

Chromium is the default browser. Change it in the code.

### Installation

Run:

1. `chmod +x index.js`
2. `npm link`
3. `clipt` to start it in any directory

Or simply:

1. `chmod +x index.js`
2. `./index.js` only in this directory
